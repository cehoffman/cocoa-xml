Cocoa-XML

github.com/cehoffman/cocoa-xml

DESCRIPTION:

Cocoa-xml provides a more ruby like interface to Cocoa’s NSXMLDocument and classes that inherit from NSXMLNode. It provides access to XPath, XQuery, and CSS selectors for searching documents.

FEATURES/PROBLEMS:

  • Cocoa-xml uses native Cocoa xml processing facilities.
  • CSS, XPath, and XQuery can be used to search a document
  • Broken HTML documents can be processed thanks to NSXMLDocument

SYNOPSIS:

  require 'cocoa-xml'

  doc = CocoaXML::HTML("http://www.google.com/search?q=cehoffman")

  doc.css('h3.r a.l').each do |link|
    puts link.text
  end

  doc.xpath('//h3/a[@class="l"]').each do |link|
    puts link.text
  end

  doc.xquery('data(//h3/a[@class="l"]')).each do |link|
    puts link
  end

REQUIREMENTS:

  • MacRuby 0.6 (development version)
  • OS X Snow Leopard (only version supported by MacRuby currently)

INSTALL:

  $ sudo gem install cocoa-xml

DEVELOPERS:

It is advised to use the current development version of MacRuby from the source tree.

CREDITS:

CSS selector support is taken from Nokogiri.
